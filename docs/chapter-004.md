
## Chapter 4. One Codebase, Multiple Platforms

* Flutter가 지원하는 플랫폼은 다음과 같다.
    * Mobile App
        * Android
        * iOS
        * iOS 앱을 만드는 경우 MacOS가 없다면 개발이 불가능하다.
    * Web Application in Modern browser
    * Desktop App
        * Windows
        * MacOS
        * Linux
        * MacOS 앱을 만드는 경우 MacOS가 없다면 개발이 불가능하다.
