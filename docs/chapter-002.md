
## Chapter 2. What is Flutter?

* 두 개의 메인 기능이 합쳐진 프레임워크이다.
    * UI 프레임워크
        * 크로스 플랫폼에서 UI를 구성한다.
    * Collection of Tools
        * CLI
        * 개발, 테스팅에 도움을 주는 도구
        * 크로스 플랫폼을 걸처 어플리케이션을 빌딩해주는 도구

* 단일 코드 베이스(single code base)로 여러 플랫폼에 어플리케이션을 배포할 수 있다.
    * Android
    * iOS
    * Windows
    * MacOS
    * Linux
    * Web

* 플러터는 단일 코드 베이스를 플랫폼에 특화된 머신 코드로 변경한다.
